## Build Order

### 1. Generate Keyframes
- Generate all 39 stills from `art/episode_01_shot_prompts.md`
- Prioritise hero shots first: S6-05, S10-02, S11-04, S12-03
- Save using exact shot IDs

### 2. Animate Keyframes
- Use `video/episode_01_motion_notes.md`
- Export each shot as 24fps MP4
- Keep most clips 3–5 seconds

### 3. Record Voices
- Record Lydrix VO first because it anchors timing
- Then key dialogue scenes: Kat/Bolt workshop, Static villain dialogue, final tag
- Record alt takes for trailer pulls

### 4. Build Sound Design
- Add ambience beds scene by scene
- Add Hyperfocus mode transformation sound as a reusable signature effect
- Add workshop room tone as a recurring comfort sound

### 5. Rough Cut Timeline
- Assemble in script order only
- Don't over-polish transitions yet
- Lock pacing before doing advanced effects

### 6. Fine Cut
- Tighten pauses
- Add glitch transitions only where story supports them
- Match music beats to S6 activation, S10 offer, S12 final line

### 7. Export Deliverables
- Full episode cut
- 60-second teaser trailer
- 15-second social cut from S6-05 + S10-02 + S12-03

## Done Definition

Episode 1 is "finished enough to show people" when:
- All 39 stills exist
- At least 20 key shots are animated
- Lydrix VO is recorded
- Dialogue exists for S3, S4, S10, S11, S12
- Music placeholders are in
- One teaser export exists
- One full rough cut exists

---

## Neurodivergent Build Rules

- Work by **scene**, not by whole episode in one sitting
- When stuck, do a **single shot** and count it as progress
- Always end a session by naming the next exact shot
- Save with exact IDs so future-you doesn't get lost
- Celebrate visible wins: hero shot done, villain reveal done, first meeting done
