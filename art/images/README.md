# Hyperfocus Mode — Art Assets

This folder contains all generated concept art and images for the Hyperfocus Mode anime project.

---

## Folder Structure

```
art/
├── images/
│   ├── characters/
│   │   ├── heroes/          ← Lydrix, Kat, Bolt
│   │   └── villains/        ← Static King, Glitch, ByteWitch, Null, Overclock, group shots
│   ├── locations/           ← Make It Work exterior/interior, Nexus City, zones
│   └── scenes/              ← Key story scenes, opening scene, trailer moments
└── ai_image_prompts.md      ← All prompts ready for Midjourney / Leonardo AI
```

---

## How to add images

1. Generate using prompts from `ai_image_prompts.md`
2. Save with the filename format below
3. Upload to the correct folder
4. Commit with a clear message
