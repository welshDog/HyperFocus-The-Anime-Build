# Hyperfocus Mode — Episode 1 Shot Image Prompt Map
## "Overload and Sparks"

**Purpose:** One prompt per shot. Use these to generate keyframe images for each scene.  
**Format:** Each shot has ID, type, description, and a ready-to-use AI image prompt.  
**Tools:** Midjourney, Leonardo AI, Adobe Firefly, Ideogram, or any anime-style generator.  
**Tip:** Keep character reference images loaded before running any shot with Lydrix, Kat, Bolt, or The Static.

## 🎨 Global Style Rules (Apply To Every Prompt)

- **Hero shots:** Add `anime key visual` to prompt
- **Scene shots:** Add `anime background art, cinematic` to prompt
- **Character close-ups:** Add `Studio Trigger aesthetic, expressive`
- **Warm scenes (Kat/workshop):** Add `Studio Ghibli warmth`
- **Villain scenes:** Add `cold, Studio Trigger villain aesthetic`
- **Lydrix POV shots:** Always include the tech-sense overlay description
- **Portrait:** `--ar 9:16`
- **Wide/scene:** `--ar 16:9`
- **Square/detail:** `--ar 1:1`
- **Always:** `--style anime`
